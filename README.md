# dotfiles
(it's my personal configuration that I use everyday, it probably won't suit you)

## Why?
> Because I just need an easy and lazy config for my daily systems. 

It's just convienient to have a setup script that will work on almost every system - arch, fedora, debian, and maybe more.

## How to run?
Either install manually by cloning the repo, installing dependencies. 

Or choose the lazy way (the right way) just enter this command `curl -fsSL https://robertplawski.pl/dot.sh | sudo sh` 

beware running scripts that you don't know, ESPECIALLY WHEN THEY REQUIRE ROOT, first check the source

It will automatically determine which package manager you are using and personalize your system for you!
